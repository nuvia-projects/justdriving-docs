# Role → Allowed actions

This section describes what each main role can do in the Just Driving platform.

## Admin

- Manage schools and settings (where applicable).
- Manage teachers:
  - Create, edit, deactivate teacher accounts.
  - Assign teachers to departments and classes.
- Manage students:
  - Create/edit student records.
  - View student progress, bookings, payments, and ratings.
- Manage classes (hold):
  - Create/edit classes (name, schedule, max students, prices, status).
  - Assign teachers and lesson plans.
  - Enroll or remove students from classes.
- Manage bookings:
  - Create/edit/cancel bookings for any student/teacher.
  - Override booking status (confirm, complete, cancel, mark no‑show).
- Finance and integrations:
  - View payment history and export data (for accounting/payroll).
  - Configure external services (payment, SMS, email templates, etc.).
- System configuration:
  - Adjust email/SMS reminder settings.
  - Manage global or school-level configuration where applicable.

## Teacher

- Schedule management:
  - View personal calendar (upcoming and past bookings).
  - Accept, reschedule, or cancel own bookings (within allowed rules).
- Student interaction:
  - View student details for assigned students/classes.
  - Mark attendance / completion for lessons.
  - Add notes on lessons or student progress.
- Class management:
  - View classes they are assigned to.
  - See the list of students in each class.
  - Optionally manage lesson plan progress (mark modules done).
- Limited booking control:
  - Create new bookings for their students (depending on school policy).
  - Propose new times/changes that require admin approval (optional, by policy).

## Student

- Account and profile:
  - View and update own profile (contact info, address, etc.).
- Bookings:
  - Browse available classes/lesson slots (where enabled).
  - Request or create bookings with a school/teacher.
  - View upcoming and past bookings, including status and details.
  - Cancel or reschedule bookings within allowed time windows.
- Payments:
  - View payment status and history for their bookings.
  - Initiate online payments where supported.
- Communication and feedback:
  - Receive notifications (email/SMS) about bookings and changes.
  - Submit ratings/reviews for schools after completed courses/lessons.
